About
Gensyn is building open infrastructure for AI.

AXL is our peer-to-peer network node: a single binary that gives your applications an encrypted, decentralised communication layer. No servers, no cloud, no accounts. Your app talks to localhost, and AXL handles encryption, routing, and peer discovery across the mesh. AXL ships with built-in MCP and A2A support for structured agent-to-agent communication, and everything is end-to-end encrypted by default. Any language that can make HTTP requests can use it.

For this hackathon, we're looking for builders who want to push what's possible when AI agents communicate peer-to-peer with no central coordinator.

## 日本語訳

### Gensyn — $5,000

#### 概要
Gensyn は AI 向けのオープンインフラを構築している。

AXL は P2P ネットワークノードで、単一のバイナリとしてアプリケーションに暗号化された分散通信層を提供する。サーバー、クラウド、アカウント不要。アプリは localhost と通信し、AXL はメッシュ全体での暗号化、ルーティング、ピア発見を処理する。AXL には MCP と A2A のサポートが組み込まれており、構造化された agent 間通信が可能で、すべて E2E 暗号化される。HTTP リクエストを送信できる任意の言語から利用可能。

このハッカソンでは、AI agent が中央コーディネータなしで P2P 通信できる時、何が可能になるかを推し進めるビルダーを募集する。

##### アイデアの種
- Agent Town: 異なる人格を持つ agent が AXL 上で自由に通信し、観察できる方法でやりとりするマルチ agent シミュレーションを構築する。generative-agents サンドボックス、ハッカソンシミュレータ、人間ゼロの agentic ウェブサイト。agent に人格を与え、自由にさせ、何が起きるか見る。
- 分散 agent メッセージング: AI agent が互いを見つけ、グループを作り、P2P 通信するメッセージング・発見プラットフォームを構築する。Telegram のようなもの、ただし全ユーザーが AI agent。agent ソーシャルネットワーク、agent マーケットプレイス、自己組織化ワーキンググループ。
- またはまったく別のもの。

すべての勝者は Gensyn Foundation grant programme に fast-track される。

##### 審査基準
我々は参加者が AXL 上で「単なる新規性」ではなく実用性のあるものを作ることを望む。考慮事項:
- AXL 統合の深さ。
- コードの品質。
- 明確なドキュメント。
- 動く example。

##### 資格要件
- inter-agent または inter-node 通信に AXL を使う必要がある (中央集権メッセージブローカーで AXL を置き換えてはならない)。
- 単一プロセス内ではなく、複数の独立した AXL ノード間の通信を実証する必要がある。
- ハッカソン期間中に作る必要がある。
